Intro to Netlify Functions

Back

Netlify Functions are backed by AWS Lambda and support running Go or Javascript.

By default all serverless functions are deployed to AWS Lambda in their us-east-1 region, with 1024MB of memory and a 10 second execution limit.

Functions can be accessed with their URLs in the format /.netlify/functions/file_name.

All HTTP methods (except HEAD) can be used to trigger the function.
